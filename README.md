# bookstore-management
# 1. Clone repository 
Cách 1: Mở visual studio 2022 
- Chọn Clone a repository
- Paste link repo https://github.com/ThankTran/bookstore-management vào ô repository location
- Chọn Clone
  
Cách 2: Dùng ternminal
- cd "đường dẫn thư mục muốn clone repo về"
- git clone https://github.com/ThankTran/bookstore-management
- Mở visual studio 2022 và chọn folder bookstore_Management vừa clone về

# 2. Chuyển sang branch cần thực hiện
